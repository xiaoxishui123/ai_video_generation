## ✅ 快速测试方案

### 测试1: 默认配置（1080p + 音频）

在工作流中添加"图片生成视频"节点：

```yaml
平台: volcengine
模型: doubao-seedance-1-5-pro-251215
图片URL: https://your-image-url.jpg
提示词: 让图片动起来
分辨率: 1080p          # ✅ 新修复：现在会生效
启用自动配音: true     # ✅ 新修复：现在会生效
时长: 5
```

**预期结果**：
```
🚀 提交图生视频任务

🏢 平台: 火山方舟
📝 模型: Seedance 1.5 Pro I2V (推荐)
🖼️ 图片: https://your-image-url.jpg
📺 分辨率: 1080p          ← 🆕 新增显示
⏱️ 时长: 5秒
🎤 配音: 自动生成         ← 🆕 新增显示
💬 描述: 让图片动起来
```

**检查要点**：
- ✅ 提示信息是否显示 "📺 分辨率: 1080p"
- ✅ 提示信息是否显示 "🎤 配音: 自动生成"
- ✅ 生成的视频分辨率是否为 1920×1080
- ✅ 生成的视频是否有声音

## 🔍 调试技巧

### 查看实际提交的参数

在 Dify 日志中查找：
```
提交的 prompt: 让图片动起来 --duration 5 --resolution 1080p --audio enable
```

### 如果视频仍然无声

可能原因：
1. 火山引擎 API 参数名称不正确（需要联系技术支持确认）
2. Seedance 1.5 Pro 模型本身不支持音频

**临时方案**：
- 改用阿里云 `wan2.6-i2v` 模型（已验证有音频功能）
